XCITE (Xerebro deCentralized InfoTech Enterprise)

A decentralized databank that provides access to millions of research repositories holding consented Xerebro datasets. Each and every client's mental activity is uploaded to an open server where doctors, researchers and people alike can access. It exists purely for research and reference purposes.

Key features

Decentralized XCITE is a free-to-use non-profit infotech platform which can be accessed by anyone anywhere.
Reliable All data(sets) on the website are proofread and cross-verified before being pushed to the database.
Secure Data on XCITE's server has a 4-level layer of encryption, eliminating any room for manipulation, e-vandalism and gaslighting users.
